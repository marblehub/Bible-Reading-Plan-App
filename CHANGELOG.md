# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - Initial Release

### Added
- Year-long Bible reading plan generated as JSON
- PyQt5 desktop application interface
- Daily reading navigation (Previous / Next)
- Automatic progress tracking (Day X / Total Days)
- Biblia API integration for Scripture retrieval
- Support for full chapters and chapter ranges
- Styled Scripture display with professional typography
- External links enabled for footnotes and citations

### Fixed
- Correct handling of multi-chapter passages using Biblia-compatible ranges

### Notes
- Requires a valid Biblia API key
- Designed for Python 3.8+
